# cursoVueJS-fundamentos

Aprende los fundamentos de Vue.js, el framework progresivo. Conoce cómo funciona la reactividad dentro de Vue.js y cómo podemos usarlo para empezar a construir aplicaciones frontend con esta librería. Descubre las ventajas de Vue junto con tu profesora Diana Martínez.

- Descubrir por qué Vue es “el framework progresivo”
- Aprender cómo Vue maneja los eventos
- Descubrir el renderizado declarativo
- Aprender cómo funciona la reactividad con Vue.js
